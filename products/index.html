Quick Overview

FX Currenecy Strength Meter expresses the value of currency for 8 different forex currency. The meter also generates 28 different entry signals for different currency pairs at every tick depicting strong buy, strong sell, etc. The strength of the currencies is shown in textual layout as well as graphical layout. The indicator uses histograms to express the value of currency at every tick. Set the period as per the financial strategy i.e 1min, 5min, 15min, 30min, 1hour, 4hour, 1day, 1week, 1month. Simple user interface to add and remove currencies.
