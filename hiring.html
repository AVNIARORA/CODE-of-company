Interview Process for software engineers

The Interview Process consists of four rounds:
  • First round consists of four basic programming questions which you can write in any programming language.
  • Second round consists of five advanced programming assignments in C.
  • Third round consists of one or two puzzles which normally takes one hour or so.
  • Fourth round is the HR discussion.
